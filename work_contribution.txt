- **Kadambari Marne**: Conducted comprehensive literature surveys and research on existing digital barter platforms, circular economies, and privacy-first architectures. **Lead: Siddhi Khandarkar**

## 3. Backend Development & System Architecture
**Lead: Kadambari Marne**

## 4. AI Moderation, Matching & Real-Time Features
**Lead: Kadambari Marne**

- Set up the Socket.IO infrastructure on the server for handling real-time, secure anonymous communication between peers.
